FitnessTracker Price Detector
About the Project
This project is a FitnessTracker Price Detector that helps predict the prices of fitness trackers available in the Indian market.

Dataset
The dataset consists of products from multiple brands, collected from e-commerce websites like Flipkart and Amazon using a web scraping technique.
Model
The project uses Linear Regression as the model for predicting the price of fitness trackers based on the features provided. To handle categorical data like brands and product types, One-Hot Encoding is applied.

Key Features
Price Prediction: Predict the price of fitness trackers based on product details.
Feature Encoding: Categorical variables such as brand are encoded using One-Hot Encoding.
Requirements
Python 3.x
Libraries: Pandas, Scikit-learn, NumPy, Matplotlib
